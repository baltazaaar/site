Отдел перспективных разработок


Уровень зарплаты

по результатам собеседования

Город

Минск, метро Спортивная

Опыт работы

1-3 года


Требования:

  • опыт (не менее 1 года) программирования микроконтроллеров на базе ядер MIPS (PIC32) или ARM (ARM7, ARM9, Cortex, SoC) или AVR (ATmega и др.) на языке программирования ANSI C (возможно C++);
  • приветствуется навык отладки аналоговых и цифровых устройств на базе различных микроконтроллеров (MCU);
  • приветствуется опыт работы с приводными интерфейсами связи: Ethertet LAN, USB (и его прикладными протоколами для классов CCID, MSC, RNDIS, VCP устройств) и иными;
  • приветствуется знание основ работы с APDU (T0) протоколом для Smartcard (SIM карты, USB Token и т.п.), согласно ISO 7816-4;
  • приветствуется знание отечественных алгоритмов шифрования/защиты (контроль целостности, подлинности) данных, согласно стандартам: ГОСТ 28147-89, СТБ 34.101.31-2011, СТБ П 34.101.45-2011;
  • приветствуется навык разработки/программирования прикладного/системного ПО для отладки программ микроконтроллеров (любая ОС, любой инструмент разработки);
  • владение английским языком на уровне чтения технической документации.

Обязанности:

  • разработка и сопровождение программ на ANSI C для PIC32MX, C/C++ на ARM Cortex-M4, программирование микроконтроллеров;
  • разработка и написание прикладных протоколов и программных обработчиков для интерфейсов Ethernet LAN (с использованием RMII), USB (с использованием ULPI), АЦП, SPI, I2C и др.

Условия:

  • интересные проекты;
  • достойная оплата труда (обсуждается индивидуально на собеседовании);
  • дружный коллектив.

Тип занятости

Полная занятость, полный день
